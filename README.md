# CarDealershipDatabase
SQL Workbench database for my Car Dealership application.

One interesting query and discovery I made during this project was the similarities and differences in SQL vs JAVA in IntelliJ. 

Big things were the Operators and seeing how they translated vs what I had been use to and how to use them during my query setups. 

Another big help to streamline my work was the thought process of how to approach the queries. For example, "Show me ____, from the ____ table, where ____ equals ____." With this mindset I found going through sections I found myself getting stuck or caught in a loop in, becoming more clear and easier to not only read but understand.
